# Cloned Repositories

This directory contains research implementations related to thinking tokens and reasoning in LLMs.

## 1. Quiet-STaR (ezelikman/quiet-star)
- **Purpose**: Implementation of "Quiet-STaR: Language Models Can Teach Themselves to Think Before Speaking".
- **Key Idea**: Generates rationales at each token to explain future text.
- **Location**: `code/quiet-star/`

## 2. Filler Tokens (rokosbasilisk/filler-tokens)
- **Purpose**: Implementation of "Let's Think Dot by Dot: Hidden Computation in Transformer Language Models".
- **Key Idea**: Investigates the use of meaningless filler tokens (e.g., '......') for hidden computation.
- **Location**: `code/filler-tokens/`

## 3. Pause Token (epfl-dlab/PauseToken)
- **Purpose**: Implementation of "Learning to Insert [PAUSE] Tokens for Better Reasoning".
- **Key Idea**: Methods for inserting [PAUSE] tokens to improve reasoning performance.
- **Location**: `code/pause-token/`
